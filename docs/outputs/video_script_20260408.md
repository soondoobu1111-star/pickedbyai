## 핵심 서사 (CEO 방향)

1. **공감 먼저** — 사용자가 실제로 AI에게 물어보는 상황으로 시작
2. **구글/네이버 vs AI 대비** — 수십 개 목록 → 2~3개만 추천
3. **"상위 0.01%만 추려서 추천"** — 나머지는 노출조차 안 됨
4. **AI 점유율은 계속 높아짐** — 위기감 구체화
5. **pickedby.ai = 해결책** — AI 인지 상태 확인 + 개선 방법 제공

## 스크립트 (30~40초)

### [0~5초] 공감 후크
```
"요즘 육아일기 앱 뭐 써?"
AI한테 물어봤더니...
딱 2개만 추천해줬습니다.
```

### [5~15초] 문제 시각화
```
(화면: 구글 검색 결과 — 수십 개 목록)
"예전엔 이랬습니다."

(화면: ChatGPT 답변 — 2~3개 + 장단점)
"이제는 이렇습니다."

AI는 상위 0.01%만 추려서 추천합니다.
나머지 제품은 노출조차 되지 않습니다.
```

### [15~25초] 위기감 → 해결책
```
AI 점유율은 앞으로 계속 높아집니다.
그렇다면 신생 기업들은 어떻게
AI에게 자사 제품을 학습시킬 수 있을까요?

(화면: pickedby.ai 데모 — 점수 나오는 화면)
pickedby.ai는 AI가 지금
당신의 제품을 인지하고 있는지 알려주고,
어떻게 하면 더 많이 인식시킬 수 있는지
해결해드립니다.
```

### [25~30초] CTA
```
pickedby.ai — 지금 무료로 확인하세요.
```

## 촬영 가이드

| 장면 | 방법 |
|------|------|
| 구글 검색 결과 화면 | 화면 녹화 (QuickTime) |
| AI 답변 화면 | ChatGPT or Perplexity 화면 녹화 |
| pickedby.ai 데모 | pickedby.ai 접속 → 제품명 입력 → 점수 출력 화면 녹화 |
| 자막 | iMovie or CapCut 자막 삽입 |

**별도 장비 불필요 — 화면 녹화 + 자막만으로 완성 가능**
