## 0. 선행 교정: AI들이 틀린 것

분석 전에 5개 AI의 오류를 먼저 정리한다. 이 오류들을 그대로 수용하면 잘못된 의사결정을 내린다.

| AI 지적 | 실제 현황 | 판단 |
|---------|-----------|------|
| "수익 모델 없음" | Free / Creator $19 / Pro $49 가격 확정 | ❌ AI 오류 |
| "Training data only" | Tavily 웹검색 기반 실시간 점수 (Step 1) | ❌ AI 오류 |
| "AI에 직접 안 물어봄" | Tavily → llama-3.1-8b + Gemini relay 파이프라인 | ❌ 부분 오류 (직접 LLM 쿼리는 Phase 2) |
| "팀 정보 없음" | THUNOVA 법인, 대표자 존재 (공개 여부는 전략적 선택) | 오류는 아니나 맥락 부재 |
| "트랙션 제로" | Reddit r/SideProject 4.7K 뷰, Product Hunt 제출 완료 | ❌ AI 오류 (스테이징 URL 기준 평가) |
| "배지 효과 미검증" | 이미 embed 코드 제공 중, A/B 데이터 수집 시작 | 부분 오류 |

**결론:** 5개 AI 모두 스테이징 URL의 프론트만 보고 분석했다. 내부 아키텍처(Tavily+llama, 결제 인프라, Phase 2 로드맵)를 몰랐다. 평가 점수 자체는 외부 관찰자 기준으로 타당하지만, 실제 격차는 AI들이 생각하는 것보다 작다.

## 1. 5개 AI 공통 합의 사항 (이견 없음)

### 1-1. 타이밍은 완벽하다
- 전원 타이밍 점수 8~9/10
- "2005년 SEO 시장" 비유 (GPT), "GEO CAGR 40.6%" (젠스파크)
- **결론:** 지금 가장 중요한 자산은 시간이다. 속도가 곧 해자다.

### 1-2. PLG/UX 구조는 정석이다
- 전원 "10초, 무가입, 즉각 결과"를 강점으로 언급
- 제미나: "마찰 없는 온보딩의 정석", 마누스: "훌륭한 리드 제너레이션 도구"
- **결론:** 상단 퍼널은 손대지 않는다. 이미 최적화되어 있다.

### 1-3. 배지가 유일한 네트워크 효과다
- 전원 배지 시스템을 유일한 differentiator로 지목
- 젠스파크: "Verisign SSL 인증서 유사 가능성", 그록: "Dynamic badge로 업그레이드 필요"
- **결론:** 배지 = 성장 엔진. 가장 먼저 검증해야 할 가설.

### 1-4. "진단"에서 "처방"으로 넘어가야 한다
- 전원 동의: 점수만으로는 재방문 이유가 없다
- 제미나: "처방과 치료", 마누스: "Actionable Optimization Engine", 그록: "llms.txt auto-generator"
- **결론:** LLM-03 (llms.txt 자동생성)이 Phase 2의 핵심 기능이다. 이 하나가 유료 전환을 만든다.

### 1-5. 기술 moat가 없다
- 전원 동의: 복제 난이도 낮음
- **결론:** 기술 moat 대신 **데이터 moat + 브랜드 moat**로 방향을 잡는다.

## 4. 수익화 경로 최적 시나리오


## 5. 기술 Moat 구축 로드맵

AI들 전원이 "기술 moat 없음"을 지적했다. 하지만 진짜 moat는 기술이 아니다.

### Moat 레이어 1: 데이터 (6개월 내 구축 가능)
```
현재: 점수 + 이메일 수집
목표: 제품 카테고리별 평균 점수 벤치마크 DB
방법: 누적 스캔 데이터 익명화 → "Notion 템플릿 카테고리 평균 AI 점수: 42점"
가치: 이 데이터는 돈으로 살 수 없다. 우리만 가진 GEO 벤치마크.
```

### Moat 레이어 2: 배지 네트워크 (현재 시작)
```
배지 embed → Gumroad/Etsy 페이지 → pickedby.ai URL 노출 → 바이럴
목표: 1,000개 배지 → 검색 엔진 + AI 학습 데이터에 "AI 추천" 신호로 인식
젠스파크 비유: Verisign SSL처럼 "이 배지 없으면 신뢰 없음" 인식 형성
```

### Moat 레이어 3: 브랜드 (2005년 SEO의 Google Analytics)
```
2005년: 누가 먼저 "웹사이트 트래픽 무료 측정 도구"를 만들었나?
2026년: 누가 먼저 "AI 가시성 무료 측정 도구"를 만들었나?
우리가 먼저다. 카테고리 브랜드 = 가장 강력한 moat.
```

### Moat 레이어 4: 플랫폼 파트너십 (P2)
```
Gumroad 공식 파트너 → "AI Visibility by pickedby.ai" 내장
LemonSqueezy 파트너십 → 셀러 대시보드 통합
이것이 실현되면 경쟁사 진입 불가.
```

## 6. 타겟 시장 재정의

### 현재 포지셔닝
"Gumroad/Etsy 디지털 크리에이터" — AI들이 "너무 좁다"고 지적

### 수정 제안: 세그먼트 래더 전략

**1단계 (지금): 인디 크리에이터** — PMF 증명용

- 그록 제안과 일치
- 이미 AI visibility를 알고, 돈을 내는 세그먼트
- "우리 SaaS가 AI에 안 보인다" = 생존 문제
- ARPU: $19~49

**3단계 (12개월): SMB / SEO Agency**
- 마누스, GPT 제안과 일치
- 화이트라벨 + 클라이언트 관리

- 대기업/Enterprise: Profound에 양보

## 8. 우리가 놓친 인사이트 (5개 AI도 언급 안 한 것)

### 인사이트 1: 배지가 AI 학습 데이터가 된다
배지가 Gumroad/Etsy 페이지에 embed되면, 그 페이지가 크롤링되고 AI 학습 데이터에 들어간다. "pickedby.ai에서 AI 추천됨"이라는 텍스트가 쌓이면, AI가 "pickedby.ai = AI 추천 신뢰 기관"으로 학습하기 시작한다. **우리 서비스가 AI 추천의 신뢰 소스 자체가 된다.**

### 인사이트 2: 점수 0인 사용자가 최고의 리드다
myths_dark1 "I got zero" 댓글 유형이 가장 강한 유료 전환 후보다. 점수 0 = 위기감 최대 = 돈을 낼 의향 최대. **점수 0 사용자에게 즉각 "개선 패키지" 오퍼를 보여주는 것이 전환 최적화의 핵심.**

### 인사이트 3: SEO 에이전시가 아니라 크리에이터 교육 플랫폼과 협업하라
GPT는 SEO Agency를 제안했지만, 더 빠른 경로는 Gumroad University, Payhip Academy, Ali Abdaal 같은 크리에이터 교육 인플루언서다. 이들 1명이 추천하면 수천 명의 크리에이터에게 즉시 도달한다. B2C보다 B2I (Business to Influencer).

### 인사이트 4: "AI에 보이지 않으면 죽는다" 공포 마케팅의 적시성
지금은 AI 검색 전환기다. 6~12개월 후면 "AI 검색 대응"이 상식이 된다. 지금의 공포감("AI가 내 제품을 모른다")은 지금만 가능한 마케팅 에너지다. **이 공포감이 유효한 지금이 가장 공격적으로 마케팅해야 할 시점.**

### 인사이트 5: Reddit 댓글이 Product Hunt 준비다
현재 r/SideProject 댓글들 (mrtrly의 기술 제안, Phull1van의 긍정 피드백)이 그대로 Product Hunt 리뷰가 된다. 댓글에 잘 대응하는 것 = Product Hunt 런치 준비. **댓글 응답 품질이 PH 성공률에 직결된다.**
